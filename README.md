> A dynamic camper van rental application built as part of the advanced React development course on Scrimba. Features structured routing, deeply nested state layouts, and an API simulation layer powered by MirageJS to manage real-time van details and user authentication.

---

## 🚀 Key Features

* **Dynamic Client-Side Routing:** Leverages modern React Router mechanics (including Layout routes, relative links, index paths, and active navigation styling states).
* **Protected Routes & Auth Flow:** Explicit redirection logic guarding the host dashboard elements from unauthorized users.
* **MirageJS API Simulation:** Intercepts outgoing HTTP requests to simulate a fully functional server environment, handling mock database operations, filtering parameters, and secure login endpoints.
* **Dashboard Control:** Nested routing structures for managing host-specific inventory, reviews, pricing, and individual van data models.

## 🛠️ Architecture & Tech Stack

* **Frontend Engine:** React with Vite
* **Package Management:** `pnpm` monorepo-ready layout strategies
* **Routing System:** React Router v6
* **Data Layer Simulation:** MirageJS
* **Styling Framework:** Custom high-contrast layout system optimized for utility and readability

## 💻 Installation and Setup

### Prerequisites
Ensure you have Node.js and `pnpm` globally configured in your environment.

### Getting Started

1. Clone the repository:
   ```bash
   git clone [https://github.com/OscarLVJ/VanLifeVbF.git](https://github.com/OscarLVJ/VanLifeVbF.git)
   cd VanLifeVbF

2. Install dependencies via pnpm:
    pnpm install

3. Spin up the local development server:
    pnpm run dev
